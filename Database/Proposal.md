# Login API

로그인 API입니다. 로그인 기능을 직접 구현하지 않고 API로 사용이 가능하게 합니다. 나중에는 ORM과 같이 제공 할 예정입니다. 데이터베이스를 직접 관리 하지 않아도 되며 간단하게 사용이 가능합니다.

## Tech Stack

- git
- Circle CI
- Python
- Flask
- Graph QL

## API Spec

- POST /Account/new
- POST /Account/database
- POST /Account/state

- POST /Database/new
- POST /Database/put
- POST /Database/get
